# JavaScript Error Handling Bonus Challenges

These three challenges are meant to reinforce:

- error objects
- `try...catch`
- `throw new Error(...)`

They start easy and get slightly more combined by the end.

## Challenges

1. `challenge-1-parse-json.js`
   Practice using `try...catch` with `JSON.parse()`.

2. `challenge-2-throw-errors.js`
   Practice validating input and throwing your own errors.

3. `challenge-3-combine-skills.js`
   Practice parsing data and throwing custom errors in one function.

## How to Use

Open each file, read the instructions, and replace the `TODO` sections with your own solution.

If you want, run a file with Node:

```bash
node challenge-1-parse-json.js
```
